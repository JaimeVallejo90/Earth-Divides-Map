# Drainage Basin World Map

Global zoomable map with thematic layers:

- Soft topographic base (land relief + ocean bathymetry).
- Tectonic plate boundaries.
- Base drainage divides (simplified version).
- Detail drainage divides (denser and very faint version).

## Open

1. Open `index.html` in your browser.
2. Use the mouse wheel or the `+/-` buttons to zoom.
3. Toggle layers in the top-left panel.

## Data Sources

- Tectonic plates: PB2002 Boundaries (`fraxen/tectonicplates` repository).
- Base divides: derived from HydroBASINS v1c (level 3, shared basin boundaries).
- Detail divides: derived from HydroBASINS v1c (level 4, shared basin boundaries).
- Base tiles: Esri public services (Ocean, World Physical, World Hillshade).
